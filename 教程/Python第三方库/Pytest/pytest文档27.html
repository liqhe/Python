设置bash完成¶

使用bash作为shell时，pytest可以使用argcomplete（https://argcomplete.readthedocs.io/）进行自动完成。为此argcomplete需要安装和启用。

使用以下命令安装argcomplete：

sudo pip install 'argcomplete>=0.5.7'

对于所有启用argcomplete的python应用程序的全局激活，运行：

sudo activate-global-python-argcomplete

对于永久（但不是全局）pytest激活，请使用：

register-python-argcomplete pytest >> ~/.bashrc

仅对argcomplete进行一次性激活pytest，请使用：

eval "$(register-python-argcomplete pytest)"
